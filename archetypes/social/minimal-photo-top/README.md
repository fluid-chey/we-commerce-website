# minimal-photo-top

Photo on the top half, bold headline + subheader grouped together on the bottom half. Combines visual impact of an image with the punch of a minimal statement.

## When to use
- Product announcements with a hero image
- Event promotions
- Any bold statement that benefits from a supporting visual

## Slots
- **photo** — 1080x540 top-half image
- **headline** — primary bold statement (96px)
- **subtext** — supporting subheader (42px)

# hero-stat-split

Split layout variant of hero-stat. Photo on the left half, text panel on the right with eyebrow, headline, body copy, and a compact 2-stat row.

## When to use
- Data-driven content that benefits from a supporting image
- Case studies, quarterly reports, product launches with visual context

## Slots
- **photo** — 540x1080 left-half image
- **eyebrow** — context label above headline
- **headline** — primary message (82px)
- **body-copy** — supporting paragraph
- **stat-1-num / stat-1-label** — first metric
- **stat-2-num / stat-2-label** — second metric

# split-photo-quote

Split layout with photo on the left and a formatted quote on the right. Includes a large decorative quotation mark, italic quote text, circular portrait avatar, and attribution with title.

## When to use
- Testimonials and customer quotes with a supporting photo
- Case study pull quotes
- Team spotlights with personal statements

## Slots
- **photo** — 540x1080 left-half image
- **quote-text** — italic pull quote
- **portrait** — 88px circular avatar next to attribution
- **attribution** — person's name
- **title** — role / company

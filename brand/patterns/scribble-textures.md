## Usage Rules

- Use sparingly as background texture or accent elements
- Apply `mix-blend-mode: screen` and low opacity (0.10-0.20)
- Best used at edges of canvas, partially off-frame
- Reference via `/api/brand-assets/serve/{name}`

```css
/* Scribble texture as background accent */
.scribble-accent {
  position: absolute;
  mix-blend-mode: screen;
  opacity: 0.12;
  pointer-events: none;
  z-index: 1;
}
```

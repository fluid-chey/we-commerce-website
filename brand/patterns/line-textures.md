Hand-drawn line textures for borders, dividers, and decorative accents.

## Available Assets

Assets: `line-049`, `line-056`, `line-059`, `line-060`, `line-061`, `line-062`, `line-067`, `line-069`, `line-072`, `line-074`, `line-077`, `line-087`, `line-167`, `line-170`, `line-173`, `line-204`, `line-233`, `line-238`, `line-288`

## Usage Rules

- Use as decorative borders or section dividers
- Apply `mix-blend-mode: screen` and low opacity (0.10-0.25) for subtle effect
- Can be rotated for horizontal or diagonal placement
- Reference via `/api/brand-assets/serve/{name}`

```css
/* Line texture as decorative divider */
.line-divider {
  position: absolute;
  mix-blend-mode: screen;
  opacity: 0.15;
  pointer-events: none;
  width: 100%;
  height: auto;
}
```

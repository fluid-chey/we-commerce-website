Consistent opacity values across all brand elements.

| Element | Opacity | Weight |
|---------|---------|--------|
| Brushstrokes | `0.10 - 0.25` | 90 |
| Circle sketches | `0.5 - 0.7` | 85 |
| Side labels | `~0.35` | 70 |
| Slide numbers | `~0.40` | 70 |
| Ghost background text | `0.04 - 0.06` | 65 |

```css
/* Opacity system */
.brushstroke     { opacity: 0.15; }  /* Range: 0.10-0.25 */
.circle-sketch   { opacity: 0.6; }   /* Range: 0.5-0.7 */
.side-label      { opacity: 0.35; }
.slide-number    { opacity: 0.40; }
.ghost-text      { opacity: 0.05; }  /* Range: 0.04-0.06 */
```
